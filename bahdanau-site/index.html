§ 01 — Premise

What kind of matrix does it form?

Bahdanau attention produces a soft alignment matrix — a continuous, row-stochastic relaxation of the discrete alignment used in classical statistical machine translation.

When a decoder generates output at time t, it doesn't read the source sequence uniformly. Instead, it produces a distribution over encoder positions describing where to look. Stack these distributions row-by-row, one per decoder step, and you get the attention matrix.

Each row is a probability distribution. Each column represents a source token. The cell at (t, i) tells you how much the decoder attended to source position i while producing target token t.

§ 02 — Formulation

The additive score function.

Unlike dot-product attention, Bahdanau uses a small feed-forward network — a single hidden layer with a tanh nonlinearity — to compare encoder and decoder states.

e(t, i) = v⊤ · tanh( W₁ · hi + W₂ · st−1 ) → Raw alignment score between decoder state s_{t-1} and encoder state h_i

Where W₁ and W₂ are learnable projection matrices and v is a learnable vector that collapses the hidden representation into a scalar score. These scores are then normalized across encoder positions via softmax:

α(t, i) = exp( e(t, i) ) / Σj exp( e(t, j) ) → Row-wise softmax turns scores into a probability distribution
ct = Σi α(t, i) · hi → Context vector: weighted sum of encoder states for decoder step t

§ 04 — Taxonomy

Three matrices, one pipeline.

The attention mechanism manufactures several distinct matrices on its way to producing a context vector. Each has a different character.

01 / SCORE

Score matrix

e ∈ ℝT_dec × T_enc

Raw, real-valued, unbounded. Each entry is a learned compatibility score between a decoder query state and an encoder key state.

02 / ATTENTION

Attention weights

α ∈ [0,1]T_dec × T_enc

Row-stochastic. Each row sums to 1 after softmax. This is the soft alignment matrix — dense, differentiable, and learned end-to-end.

03 / CONTEXT

Context bundle

C ∈ ℝT_dec × d_h

The result of α · H — one context vector per decoder step. Fed into the decoder alongside its own hidden state to predict the next token.

§ 05 — Implementation

Bahdanau, in PyTorch.

A compact reference implementation showing how the three matrices are assembled in code.

bahdanau.py PyTorch 
import torch
import torch.nn as nn

class BahdanauAttention(nn.Module):
    def __init__(self, enc_dim, dec_dim, attn_dim):
        super().__init__()
        self.W1 = nn.Linear(enc_dim, attn_dim, bias=False)
        self.W2 = nn.Linear(dec_dim, attn_dim, bias=False)
        self.v  = nn.Linear(attn_dim, 1, bias=False)

    def forward(self, H_enc, s_prev):
        # H_enc : (batch, T_enc, enc_dim)
        # s_prev: (batch, dec_dim)

        # 1. Score matrix e : (batch, T_enc)
        proj = torch.tanh(self.W1(H_enc) + self.W2(s_prev).unsqueeze(1))
        e = self.v(proj).squeeze(-1)

        # 2. Attention weights alpha : (batch, T_enc) — row-stochastic
        alpha = torch.softmax(e, dim=-1)

        # 3. Context vector c : (batch, enc_dim)
        c = torch.bmm(alpha.unsqueeze(1), H_enc).squeeze(1)

        return c, alpha
