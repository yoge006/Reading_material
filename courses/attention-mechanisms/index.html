Lesson 01 — NLP · Attention

Bahdanau Attention

The Alignment Matrix · Bahdanau et al., 2014
Read time ~4 min lesson
Format Visual Essay
Paper ICLR 2015

Standard seq2seq models compress an entire input sequence into a single fixed-length vector — a bottleneck that loses information for long sentences. Bahdanau attention replaces this with a dynamic context vector recomputed at each decoder step.

What you'll learn
  • Why the fixed-length encoder vector fails on long sequences
  • How alignment scores are computed using a feed-forward network
  • How softmax turns scores into attention weights αᵢⱼ
  • How the context vector cᵢ is assembled as a weighted sum of encoder states
  • How to read the attention alignment matrix as a diagnostic tool

Four key concepts.

01 — Problem
The Bottleneck

A vanilla seq2seq encoder compresses the full input into one vector c. For long sequences, early tokens are forgotten — the decoder has no direct path back to them.

02 — Score Function
Alignment Scores

For each decoder step i and encoder state j, compute a score:

e_ij = vᵀ tanh(W₁s_{i-1} + W₂h_j)

A small feed-forward network that asks: "how well does source position j match decoder state i?"

03 — Weights
Attention Weights αᵢⱼ

Pass the scores through softmax to get a probability distribution over source positions. These are the attention weights — they sum to 1 for each decoder step.

04 — Context
Context Vector cᵢ

The context vector is a weighted sum of all encoder hidden states:

c_i = Σⱼ α_ij · h_j

A fresh, query-specific summary recomputed at each decoder step.

The alignment matrix.

Stacking all attention weight vectors [α₁, α₂, …, αₘ] across decoder steps produces an alignment matrix — a grid showing which source tokens the model focused on for each output token.

For a well-trained translation model, this matrix should approximate a diagonal pattern: the first output word should attend to the first input word, the second to the second, and so on — with soft deviations for syntactic reordering.

PyTorch sketch.

attention.py — Score function + context vector
import torch
import torch.nn as nn
import torch.nn.functional as F

class BahdanauAttention(nn.Module):
    def __init__(self, hidden_dim):
        super().__init__()
        self.W1 = nn.Linear(hidden_dim, hidden_dim)  # decoder state
        self.W2 = nn.Linear(hidden_dim, hidden_dim)  # encoder states
        self.v  = nn.Linear(hidden_dim, 1)

    def forward(self, s_prev, h):
        # s_prev: (batch, hidden_dim)
        # h:      (batch, src_len, hidden_dim)
        s = self.W1(s_prev).unsqueeze(1)  # (batch, 1, H)
        e = self.v(torch.tanh(s + self.W2(h)))  # (batch, src_len, 1)
        alpha = F.softmax(e.squeeze(-1), dim=-1)  # (batch, src_len)
        context = (alpha.unsqueeze(-1) * h).sum(dim=1)  # (batch, H)
        return context, alpha
